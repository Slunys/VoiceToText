# VoiceToLun 🎙️

Удобное Windows-приложение для голосового ввода текста. Работает полностью в офлайне на базе нейросети `faster-whisper`. 
Вы удерживаете назначенную клавишу, диктуете текст, и программа моментально впечатывает его в любое активное окно (Блокнот, мессенджер, браузер).

## Особенности ✨
* **Полная приватность:** Никаких облачных API. Аудио обрабатывается локально на вашем процессоре/видеокарте.
* **Мультиязычность:** Отлично распознает русскую речь и корректно вписывает английские ИТ-термины прямо посреди предложения.
* **Свой интерфейс:** Красивое окно настроек (на базе `customtkinter`) и удобный фоновый режим — программа сворачивается в системный трей.
* **Кастомные бинды:** Кнопку для активации записи можно легко переназначить в самом приложении. Поддерживаются дополнительные клавиши с макро-клавиатур (например, F13-F24).

## Установка и запуск из исходников 🛠️

1. Склонируйте репозиторий:
```bash
git clone [https://github.com/Slunys/VoiceToText.git](https://github.com/Slunys/VoiceToText.git)
cd VoiceToText
```

2. Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

3. Запустите программу:
```bash
python main.py
```

(При первом запуске скрипт автоматически скачает нужную модель в папку whisper_models).

Инструкция по сборке в .exe 📦
Для самостоятельной сборки проекта в исполняемый файл используется PyInstaller.
Выполните следующую команду в терминале:
```bash
pyinstaller --noconsole --name "VoiceToLun" --collect-all faster_whisper --collect-all ctranslate2 main.py
```
Важно: После завершения сборки обязательно скопируйте папку whisper_models (с весами нейросети) в итоговую папку dist/VoiceTyping/, чтобы она находилась рядом с .exe файлом.
