# VoiceToText 🎙️

Мощное и удобное Windows-приложение для голосового ввода текста, работающее полностью локально. Позволяет диктовать текст голосом и моментально вставлять его в любое активное окно (мессенджеры, код-редакторы, браузеры, текстовые документы).

Работает на базе нейросети **[faster-whisper](https://github.com/SYSTRAN/faster-whisper)**, что обеспечивает высочайшую скорость и точность распознавания без задержек и подключения к интернету.

---

## 🌟 Основные возможности

* **Полная приватность (Offline):** Никаких API-ключей, подписок и отправки данных в облако. Вся обработка происходит исключительно на вашем железе.
* **Умная мультиязычность:** Идеально понимает русскую речь и корректно вписывает английские ИТ-термины, названия и сленг прямо посреди русского предложения.
* **Два режима диктовки:**
* *Push-to-Talk:* Удерживайте кнопку — говорите — отпускаете кнопку (текст моментально печатается).
* *Непрерывная запись:* Нажмите кнопку дважды подряд, чтобы надиктовать длинный текст. Нажмите еще раз для остановки.


* **Моментальная вставка:** Текст появляется на экране мгновенно целиком (через буфер обмена Windows), а не печатается по одной букве. Ваш предыдущий буфер обмена при этом сохраняется!
* **Кастомные бинды:** Горячую клавишу можно переназначить прямо в приложении. Поддерживаются дополнительные F-клавиши (F13-F24), что идеально подходит для владельцев макро-клавиатур.
* **Интеграция с Windows:**
* Удобный и современный темный интерфейс на базе `CustomTkinter`.
* Фоновый режим: программа сворачивается в системный трей (возле часов) и не мешает работе.
* Тумблер автоматического запуска вместе с Windows.



---

## 🛠️ Технические требования

* **ОС:** Windows 10 / 11
* **Python:** 3.8+ (для запуска из исходников)
* **Железо:** Современный многоядерный процессор (например, Intel Core i5 12-го поколения или аналог от AMD). Программа оптимизирована для быстрой работы на CPU (`compute_type="int8"`).

## 🚀 Установка и запуск из исходников

1. **Склонируйте репозиторий:**
```bash
git clone https://github.com/Slunys/VoiceToText.git
cd VoiceToText
```


2. **Создайте виртуальное окружение и активируйте его:**
```bash
python -m venv venv
venv\Scripts\activate
```


3. **Установите необходимые зависимости:**
```bash
pip install -r requirements.txt
```


4. **Запустите программу:**
```bash
python main.py
```


> 💡 **Примечание:** При самом первом запуске программа автоматически скачает веса нейросети (модель `medium`, около 1.5 ГБ) в папку `whisper_models`. Это займет несколько минут в зависимости от скорости интернета. Все последующие запуски будут моментальными.

## 📦 Сборка в автономный .exe файл

Вы можете скомпилировать проект в один исполняемый файл, чтобы использовать его без установленного Python. Для этого используется `PyInstaller`.

1. Убедитесь, что вы находитесь в активированном виртуальном окружении.
2. Выполните команду сборки:
```bash
pyinstaller --noconsole --name "VoiceTyping" --collect-all faster_whisper --collect-all ctranslate2 main.py
```


3. После завершения сборки перейдите в появившуюся папку `dist/VoiceTyping/`.
4. **ОБЯЗАТЕЛЬНО:** Скопируйте папку `whisper_models` из корня проекта внутрь `dist/VoiceTyping/`, чтобы она лежала рядом с файлом `VoiceTyping.exe`.
5. Теперь папку `VoiceTyping` можно переместить в любое удобное место на компьютере и запускать программу через `.exe` файл!

## ⚙️ Использование

1. Запустите приложение. Вы увидите статус **"🟢 Готово к работе"**.
2. В интерфейсе нажмите **"Изменить кнопку"** и нажмите любую удобную клавишу (или кнопку на макро-паде).
3. При необходимости включите тумблер **"Автозапуск с Windows"**.
4. Нажмите крестик ❌ на окне — программа свернется в трей и продолжит работу в фоне.
5. Поставьте курсор в любое текстовое поле, зажмите выбранную кнопку, продиктуйте текст и отпустите. Текст появится моментально!

## 🧩 Используемые технологии

* [faster-whisper](https://github.com/SYSTRAN/faster-whisper) — ядро распознавания речи.
* [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) — современный графический интерфейс.
* [pystray](https://github.com/moses-palmer/pystray) — работа с системным треем Windows.
* [pynput](https://github.com/moses-palmer/pynput) — глобальный перехват нажатий клавиатуры.
* [sounddevice](https://python-sounddevice.readthedocs.io/) — захват аудиопотока с микрофона.
